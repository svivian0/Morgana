# Morgana

Morgana é uma inteligência artificial desenvolvida em Python para uso pessoal, servindo como base para um assistente pessoal, chatbot e automação de tarefas.


---

## Tecnologias Utilizadas

- **Python** (Linguagem base)
- **Streamlit** (Interface Web)
- **Pandas** (Leitura e manipulação de dados)
- **Scikit-Learn** (Treinamento do modelo de Regressão Linear)
- **Matplotlib** (Visualização e plotagem de dados)

---

## 📂 Estrutura do Projeto

```text
├── machinel/
│   ├── morgana.ipynb   # Notebook de experimentos e análise de dados
│   └── teste.csv       # Dataset base com dados de treino (X, Y)
├── LICENSE.txt         # Licença do projeto (MIT Modificada)
├── README.md           # Este arquivo de documentação
├── main.py             # Arquivo principal do aplicativo Streamlit
└── requirements.txt    # Dependências do projeto
```

---

## 🔧 Como Instalar e Executar

### 1. Pré-requisitos
Certifique-se de ter o Python 3 instalado em seu computador.

### 2. Instalar as dependências
Abra o terminal na pasta do projeto e execute o comando abaixo para instalar as bibliotecas necessárias:

```bash
pip install -r requirements.txt
```

### 3. Executar a aplicação
Morgana ainda nao possui uma interfaçe executavel, ainda estou desenvolvendo ela.
